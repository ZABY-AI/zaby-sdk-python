# MCP Client

The Model Context Protocol (MCP) client manages MCP servers, their installations, tool access, credential bindings, and auth policies.

```python
zaby.mcp  # → McpClient
```

## Server management

### Create

<ParamField body="name" type="string" required>Display name</ParamField>
<ParamField body="slug" type="string" required>URL-friendly identifier</ParamField>
<ParamField body="type" type="string" required><code>"EXTERNAL"</code> or <code>"INTERNAL"</code></ParamField>
<ParamField body="endpoint" type="string" required>MCP server endpoint URL</ParamField>
<ParamField body="description" type="string">Human-readable description</ParamField>

```python
server = await zaby.mcp.create_server({
    "name": "Web Search",
    "slug": "web-search",
    "type": "EXTERNAL",
    "endpoint": "https://mcp.search.io/sse",
})
```

### Get

```python
server = await zaby.mcp.get_server(server_id)
```

### Update

```python
await zaby.mcp.update_server(server_id, {
    "name": "Updated Name",
    "endpoint": "https://mcp.new-endpoint.io/sse",
})
```

### Discover tools

Probe the server for available tools without installing:

```python
tools = await zaby.mcp.discover_tools(server_id)
```

## Tool catalog

Browse available MCP tools across all registered servers:

```python
catalog = await zaby.mcp.list_catalog()
```

## Installations

### Install

```python
installation = await zaby.mcp.install_server({
    "serverId": server_id,
    "name": "Search Prod",
    "config": {"api_key": "sk-..."},
})
```

<ParamField body="serverId" type="string" required>Server to install</ParamField>
<ParamField body="name" type="string" required>Installation name</ParamField>
<ParamField body="config" type="dict">Server-specific configuration</ParamField>

### List installations

```python
installations = await zaby.mcp.list_installations()
```

### Update installation

```python
await zaby.mcp.update_installation(installation_id, {
    "config": {"api_key": "new-key"},
})
```

### Revoke installation

```python
await zaby.mcp.revoke_installation(installation_id)
```

---

## Tool management

### List installation tools

```python
tools = await zaby.mcp.list_installation_tools(installation_id)
```

### Update tool policy

```python
await zaby.mcp.update_tool_policy(installation_id, tool_id, {
    "allowed": True,
    "requiredRole": "admin",
})
```

### Preflight invocation

Validate a tool invocation without executing it:

```python
result = await zaby.mcp.preflight_invocation(installation_id, tool_name, {
    "arguments": {"query": "test"},
})
```

### Invoke tool

```python
result = await zaby.mcp.invoke_tool(installation_id, tool_name, {
    "arguments": {"query": "What's the weather?"},
})
```

## Full workflow

```python
import asyncio
from zaby import Zaby

async def main():
    zaby = Zaby(api_key="zaby_pk_...")

    server = await zaby.mcp.create_server({
        "name": "Weather",
        "slug": "weather",
        "type": "EXTERNAL",
        "endpoint": "https://weather-mcp.example.com/sse",
    })

    installation = await zaby.mcp.install_server({
        "serverId": server["id"],
        "name": "Weather Prod",
        "config": {"api_key": "sk-..."},
    })

    await zaby.mcp.invoke_tool(installation["id"], "get_forecast", {
        "arguments": {"city": "London"},
    })

asyncio.run(main())
```
